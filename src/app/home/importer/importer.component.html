SOLUTION

The final solution is limited interactive auto-forwarding set of provider coverage maps with supporting details shown in a attractive left side flyout. The users can control which region they are viewing from a pinned navigation pane at the bottom of the screen. The map also goes for the wow factor by displaying all of the providers spread across the world at once when the map first loads.

CHALLENGES AND LESSONS LEARNED

The biggest challenge for this project was working without access to the users. As a UX professional the best way to set design direction and to defend a design decisions is to base them on strong user research data. There will always be a balance between business, technology and UX, but it is very difficult to strike the correct balance when UX is advocating for a user which has not been researched.

The next challenge was facilitating concenious between the stakeholders. This was the primary theme of the project as I went through each iteration; providing a means for the stakeholders to envision the solution and come to an agreement on just what that solution should be. Being able to present the stakeholders with clickable mockups each iteration went a long way to help them visualize the potential solutions, but I know I could have driven conversations better.

In the end I believe that this project resulted in a solution which met the goals we set out to achieve and did so within the timeframe allowed to complete it.
